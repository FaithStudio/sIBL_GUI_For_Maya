Changes
=======

2.0.2
-----

-  Ensured that space in executable name doesn't prevent application launch in Mac Os X and Linux.

-----

-  Fixed executable path filtering rule on Mac Os X.

2.0.0
-----

-  Code refactor for sIBL_GUI 4.0.0.
